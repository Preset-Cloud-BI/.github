## Fixing Common Preset Cloud Issues

- Preset Cloud login not working? Confirm the correct workspace URL, SSO provider, user role, and invitation status before resetting access.
- Dashboard loading slowly? Review dataset queries, warehouse performance, chart limits, and Preset Cloud dashboards filters that may be scanning too much data.
- SQL results look unexpected? Recheck database permissions, schema selection, and Preset Cloud SQL editor query logic before publishing charts.
- Embedded view not appearing? Validate Preset Cloud embedded analytics configuration, application permissions, allowed domains, and token handling.
- Unsure where to start? Use Preset Cloud GitHub resources, Preset Cloud documentation, and Preset Cloud tutorials to compare examples, setup notes, and workflow patterns.
